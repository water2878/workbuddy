# 畅腾升降桌型号判断规则

## 一、判断流程（按优先级）

### 第1步：判断管型（最直观）

| 管型      | 特征描述       | 对应型号                            |
| ------- | ---------- | ------------------------------- |
| **方管**  | 四方形立柱，棱角分明 | F4200、F4212、T412、T423、T524、T523 |
| **椭圆管** | 椭圆形立柱，圆润外观 | T621、F4404、F6201                |
| **圆管**  | 正圆形立柱（较少见） | 老款或定制款                          |

**判断要点**：看桌腿截面形状

- 方管 = 正方形/长方形截面
- 椭圆管 = 椭圆形状，长轴短轴明显

***

### 第2步：判断电机数

| 判断点      | 单电机          | 双电机       |
| -------- | ------------ | --------- |
| **桌腿结构** | 单立柱或联动双立柱    | 两条完全独立的立柱 |
| **手控器**  | 简单上下键，无记忆或1档 | 2档或4档记忆功能 |
| **升降速度** | 25mm/s       | 40mm/s    |
| **承重能力** | 80Kg         | 120Kg     |
| **价格区间** | 400-600元     | 700-1600元 |

**判断要点**：

- 看控制器 → 有记忆键 = 双电机概率大
- 看立柱独立性 → 两根立柱完全分开 = 双电机

***

### 第3步：判断节数

| 节数     | 升降范围       | 特征          | 对应型号                       |
| ------ | ---------- | ----------- | -------------------------- |
| **2节** | 700-1200mm | 升降柱只有两段     | T524、T523、T724             |
| **3节** | 600-1250mm | 升降柱有三段，行程更大 | T621、F4200、F4212、T412、T423 |

**判断要点**：看立柱伸缩段数，3节通常更细长

***

### 第4步：判断安装方式

| 方式     | 特征        | 说明          |
| ------ | --------- | ----------- |
| **正装** | 粗段在下，细段在上 | 稳定性好，常见     |
| **倒装** | 细段在下，粗段在上 | 较少见，T523是倒装 |

**判断要点**：看立柱粗细分布

***

### 第5步：判断脚型

| 脚型      | 特征     | 对应型号        |
| ------- | ------ | ----------- |
| **平脚**  | 脚底平直   | F4200、F4212 |
| **加宽脚** | 脚底加宽设计 | T621        |
| **弓形脚** | 弧形脚底   | 部分老款        |
| **带轮脚** | 带万向轮   | 定制款         |

***

## 三、常见判断错误

| 错误            | 原因      | 正确做法         |
| ------------- | ------- | ------------ |
| 把方管看成椭圆管      | 光线或角度问题 | 多看几个角度，看截面形状 |
| 把单电机看成双电机     | 都有两根立柱  | 看控制器是否有记忆功能  |
| 把3节看成2节       | 升降柱收缩状态 | 看完全展开时的段数    |
| 混淆F4200和F4212 | 外观几乎一样  | 看铭牌或询问客户具体型号 |

***

## 四、不确定时的处理

1. **先确认管型** → 方管/椭圆管/圆管
2. **询问客户** → "您方便拍下桌腿侧面吗？"
3. **询问型号** → "您知道具体型号吗？"
4. **给出范围** → "这款应该是F4200或F4212系列"
