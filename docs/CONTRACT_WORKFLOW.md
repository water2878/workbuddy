# 合同生成流程详解

> **文档版本**: 2026-05-14  
> **适用场景**: CLAW智能客服系统合同生成

---

## 一、合同生成触发条件

客户明确表示"要合同"、"看看合同"或信息收齐后，主动提出生成合同。

---

## 二、信息收集清单（必须齐全）

| 信息项 | 说明 | 示例 |
|--------|------|------|
| **公司全称** | 营业执照上的完整名称 | 佛山挖掘机公司 |
| **联系人** | 实际对接人姓名 | 王老板 |
| **联系电话** | 手机号码 | 15633668899 |
| **收货地址** | 详细地址 | 佛山市西樵山旁边 |
| **产品型号** | 升降桌型号 | T523 |
| **数量** | 多少套 | 30套 |
| **颜色** | 钢架颜色 | 白色 |
| **面板规格** | 尺寸+厚度+级别+颜色 | 1600*800*18mm E0级白色 |
| **单价** | 成交价 | 660元 |
| **总价** | 数量×单价 | 19800元 |

**⚠️ 禁止**: 信息不全时生成合同

## 三、合同生成流程

### 步骤1: 确认客户要合同

客户说"看看合同吧"、"发合同"等 → 确认信息齐全 → 开始生成

### 步骤2: 调用合同生成API

**API端点**: `POST http://127.0.0.1:5032/api/contracts/generate`

**请求参数**:
```json
{
  "company_name": "佛山挖掘机公司",
  "customer_contact": "王老板",
  "customer_phone": "15633668899",
  "customer_address": "佛山市西樵山旁边",
  "products": [
    {
      "model": "T523",
      "quantity": 30,
      "unit_price": 660,
      "subtotal": 19800,
      "frame_color": "白色"
    },
    {
      "model": "E0",
      "quantity": 30,
      "unit_price": 265,
      "subtotal": 7950,
      "frame_color": "白色",
      "frame_size": "1600*800*18mm"
    }
  ],
  "customer_nickname": "健康办公研究社",
  "delivery_date": "",
  "payment_terms": "",
  "notes": "面板：E0级白色1600*800*18mm"
}
```

**命令行调用**:
```bash
python generate_contract.py \
  --customer "佛山挖掘机公司" \
  --contact "王老板" \
  --phone "15633668899" \
  --address "佛山市西樵山旁边" \
  --model "T523" \
  --quantity 30 \
  --price 660 \
  --panel "E0白色1600*800*18mm" \
  --color "白色"
```

### 步骤3: 处理API响应

**成功响应**:
```json
{
  "code": 200,
  "data": {
    "contract_id": "CT20260514144456",
    "created": true,
    "status": "pending"
  },
  "message": "合同已推送云端生成并进入审批流程",
  "success": true
}
```

**关键字段**:
- `contract_id`: 合同号（如 CT20260514144456）
- `status`: pending（待审批）
- `created`: true（创建成功）

### 步骤4: 通知客户

**回复内容**:
```
合同已生成！合同号：CT20260514144456。
30套T523白色钢架+E0白色面板1600*800*18mm，660元/套，总价19800元。
已提交审批，通过后发给您。
```

## 四、合同状态流转

```
生成 → 待审批(pending) → 审批通过(approved) → 已发送(sent)
                ↓
            审批拒绝(rejected)
```

| 状态 | 说明 | 操作 |
|------|------|------|
| **pending** | 已生成，等待审批 | 通知客户"已提交审批" |
| **approved** | 审批通过 | 下载PDF发送给客户 |
| **sent** | 已发送给客户 | 完成 |
| **rejected** | 审批拒绝 | 查看原因，修改后重新生成 |

## 六、常见问题处理

### Q1: 合同生成失败

**可能原因**:
- 产品信息不全
- 价格计算错误
- 网络问题

**处理**: 检查参数 → 重试 → 如仍失败，记录错误日志

### Q2: 客户信息有误

**处理**: 不要修改原合同 → 重新生成新合同 → 说明"合同号XXX作废，新合同号YYY"

### Q3: 审批长时间不通过

**处理**: 联系审批人（A畅腾升降桌09-15989997886）询问进度
