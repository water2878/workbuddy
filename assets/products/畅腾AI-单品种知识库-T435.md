## 四、安装与使用档案

### 4.1 安装方式

- **安装难度**：高（预计耗时 60-90 分钟）
- **安装方式**：建议上门安装（含在售价内）

**安装工具清单：**
1. 专业安装工具组
2. 水平仪
3. 卷尺
4. 直角尺

### 4.2 安装步骤

1. 上门勘测，确认墙角为90度直角，地面平整
2. 取出所有配件，核对三脚底脚、立柱、L型横梁、台托、螺丝包等
3. 将三根立柱与底脚连接，用水平仪调节至水平
4. 组装L型横梁，确保转角90度精准
5. 将横梁与三脚立柱组装，根据桌面尺寸调节并固定
6. 安装台托至指定位置，确保台托贴合牢固
7. 连接三电机与手控器线路，配置同步控制系统
8. 检查线路后接通电源，测试升降功能和水平同步

### 4.3 日常操作指南

| 操作 | 方法 |
|------|------|
| 升降操作 | 按手控器上升 / 下降键，到达目标高度松开即可 |
| 高度记忆 | 调节至目标高度后，长按手控器对应记忆按键 3 秒保存 |
| 童锁功能 | 上下键 + 1 键长按 10 秒锁定；重复操作解锁 |
| 复位操作 | 断电重启或长按复位键恢复出厂设置 |
| 水平校准 | 如桌面不水平，联系售后进行专业校准 |

### 4.4 保养维护方法

- **日常清洁**：用干抹布或微湿抹布擦拭
- **润滑保养**：每隔 3 个月给立柱伸缩部位润滑
- **定期检查**：每月检查各连接螺丝是否松动
- **水平检查**：每季度检查桌面水平度
- **注意事项**：避免超重放置，遵循工作休整规则

### 4.5 常见操作误区与规避

| 误区 | 正确做法 |
|------|----------|
| 自行安装L型桌架 | 建议由专业团队上门安装 |
| 忽略三脚水平度 | 安装时必须确保三脚完全水平 |
| 使用单路电源供电 | 必须使用双路供电或16A插座 |
| 忽略电机休整规则 | 单次升降后间隔 18 分钟再操作 |
