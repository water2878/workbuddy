## 四、安装与使用档案

### 4.1 安装方式

- **安装难度**：专业级（必须专业团队安装）
- **安装方式**：必须上门安装（含在售价内）

**安装工具清单：**
1. 专业安装工具组
2. 专业水平仪
3. 卷尺
4. 激光水平仪
5. 电动工具套装

### 4.2 安装步骤

1. 专业团队上门勘测，确认地面平整且承重足够（≥300kg）
2. 确认电源线路符合要求（220V/50Hz，16A插座×2或独立线路）
3. 取出所有配件，核对四脚底脚、立柱、横梁、台托、螺丝包等
4. 将四根立柱与底脚连接，用水平仪调节至水平
5. 组装横梁系统，确保四脚布局精准
6. 将横梁与四脚立柱组装，根据桌面尺寸调节并固定
7. 安装台托至指定位置，确保台托贴合牢固
8. 连接四电机与手控器线路，配置四脚同步控制系统
9. 检查线路后接通电源，测试升降功能和四脚水平同步

### 4.3 日常操作指南

| 操作 | 方法 |
|------|------|
| 升降操作 | 按手控器上升 / 下降键，到达目标高度松开即可 |
| 高度记忆 | 调节至目标高度后，长按手控器对应记忆按键 3 秒保存 |
| 童锁功能 | 上下键 + 1 键长按 10 秒锁定；重复操作解锁 |
| 复位操作 | 断电重启或长按复位键恢复出厂设置 |
| 水平校准 | 如桌面不水平，联系售后进行专业校准 |
| USB充电 | 使用桌面USB口为设备充电 |

### 4.4 保养维护方法

- **日常清洁**：用干抹布或微湿抹布擦拭
- **润滑保养**：每隔 3 个月给立柱伸缩部位润滑
- **定期检查**：每月检查各连接螺丝是否松动
- **水平检查**：每季度检查桌面水平度
- **专业保养**：建议每年进行一次专业检测保养
- **注意事项**：避免超重放置，遵循工作休整规则

### 4.5 常见操作误区与规避

| 误区 | 正确做法 |
|------|----------|
| 自行安装4410C | 必须由专业团队上门安装 |
| 使用普通插座供电 | 必须使用16A插座×2或独立线路 |
| 地面承重不足 | 确保地面承重≥300kg |
| 忽略四脚水平度 | 安装时必须确保四脚完全水平 |
| 忽略电机休整规则 | 单次升降后间隔 18 分钟再操作 |
