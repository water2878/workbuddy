## 二、重量计算公式

### 面板重量计算
```
面板重量(kg) = ⌈长(mm) × 宽(mm) × 厚度(mm) / 1000000 + 2⌉ （向上取整）
```

### 计算示例
| 面板尺寸 | 厚度 | 计算过程 | 面板重量 |
|---------|------|---------|---------|
| **1200×600** | 18mm | ⌈1200×600×18/1000000+2⌉ = ⌈14.96⌉ | **15kg** |
| **1200×600** | 25mm | ⌈1200×600×25/1000000+2⌉ = ⌈20⌉ | **20kg** |
| **1400×700** | 18mm | ⌈1400×700×18/1000000+2⌉ = ⌈19.64⌉ | **20kg** |
| **1400×700** | 25mm | ⌈1400×700×25/1000000+2⌉ = ⌈26.5⌉ | **27kg** |
| **1600×700** | 18mm | ⌈1600×700×18/1000000+2⌉ = ⌈22.16⌉ | **23kg** |
| **1600×700** | 25mm | ⌈1600×700×25/1000000+2⌉ = ⌈30⌉ | **30kg** |
| **1600×800** | 18mm | ⌈1600×800×18/1000000+2⌉ = ⌈25.04⌉ | **26kg** |
| **1600×800** | 25mm | ⌈1600×800×25/1000000+2⌉ = ⌈34⌉ | **34kg** |
| **1800×800** | 18mm | ⌈1800×800×18/1000000+2⌉ = ⌈27.92⌉ | **28kg** |
| **1800×800** | 25mm | ⌈1800×800×25/1000000+2⌉ = ⌈38⌉ | **38kg** |
